# 小红书图文生成器

一个纯前端网页小工具，用于把输入文本转换成适合发布到小红书的图片。

## 功能

- 保留输入文本的换行、空行、列表和缩进。
- 支持标题输入，标题会显示在导出图片中。
- 导出文件会以标题命名。
- 支持长图输出。
- 支持按固定高度分段输出多张图片。
- 支持选择固定保存目录，并在支持的浏览器中记住该目录。
- 移动端不支持固定目录时会隐藏目录选择，并回退到浏览器原生下载。
- 导出时自动避开重名文件，不覆盖已有文件。
- 导出完成后显示醒目的完成提示框。
- 支持模板、字体、宽度、字号、行距、边距和分段高度设置。
- 页脚展示版权、作者和 GitHub 开源仓库信息。
- 使用 Canvas 在浏览器内生成 PNG，不需要后端或构建流程。

## 使用方式

直接用浏览器打开 `index.html`。

也可以在项目目录启动任意静态服务，例如：

```bash
python -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 文件命名

- 长图：`标题.png`
- 长图重名：`标题(1).png`、`标题(2).png`
- 分段：`标题-01.png`、`标题-02.png`
- 分段重名：`标题(1)-01.png`、`标题(1)-02.png`

如果浏览器不支持固定目录写入，桌面端会优先回退到保存窗口；移动端会隐藏目录选择并使用浏览器原生下载。

固定目录选择依赖浏览器的 File System Access API。建议使用最新版 Chrome 或 Edge，并直接打开页面，不要放在嵌入式预览窗口里。
