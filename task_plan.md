## 关键问题

1. 是否需要后续支持自定义背景图或图片贴纸？本期不做。
2. 是否需要批量下载打包 ZIP？本期先逐张下载，后续可增强。
3. 极长文本是否强制使用分段模式？实现时需要根据 Canvas 限制做提示。

## 已做决策

| 决策 | 理由 |
|------|------|
| 采用纯 HTML/CSS/JS | 符合“不需要复杂架构”的要求，直接打开或静态服务即可使用 |
| 使用 Canvas 渲染导出 | 能稳定生成 PNG，并控制文字排版和切图 |
| 默认 1080px 宽度 | 贴近小红书常见图文尺寸 |
| 提供长图和固定分段两种模式 | 对应用户明确提出的两种输出方式 |
| 创建 `PROJECT_DESIGN.md` | 方便后续跟踪需求、范围和验收 |

## 遇到的错误

| 错误 | 尝试次数 | 解决方案 |
|------|---------|---------|
| 初次并行读取部分文件时遇到沙箱读取限制 | 1 | 对必要技能文件使用审批后的读取方式，并继续在项目目录内工作 |
| 当前目录不是标准 Git 仓库 | 1 | 不依赖 Git 状态作为进度依据，改用文件检查 |
| `agent-browser` 默认启动 CDP 通道失败 | 2 | 指定本机 Chrome 路径、`--no-sandbox`、`--disable-gpu`、独立 session，并使用 `agent-browser.cmd` batch 验证 |
